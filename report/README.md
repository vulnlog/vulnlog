# HTML Report Dependency

1. Download the Datatables dependencies: https://datatables.net/download/ and select:
    * Styling
        * Bulma
    * Packages:
        * jQuery
        * Bulma
    * DataTables core:
        * DataTables
    * Extensions:
        * Buttons
        * SearchBuilder
2. Copy the datatables.min.* files into _resources_.
